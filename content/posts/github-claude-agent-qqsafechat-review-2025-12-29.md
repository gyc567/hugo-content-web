### 项目描述
基于 UI Automation（UIA）的 QQ 聊天机器人完整实现，支持 AI 自动回复与表情包发送。 只需配置 API Key 并绑定 QQ 输入框、聊天列表和发送按钮即可使用。 本项目对 QQ 实现零注入、零内存读写，是目前最安全的 QQ 聊天机器人实现之一。
